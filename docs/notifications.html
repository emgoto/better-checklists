Users with commenting permissions enabled

To enable this feature, at least one user needs to have enabled comments to be posted using their account.


If you are the only user that has enabled commenting, @mentions by Checklist+ will always use your account. This effectively means that you will not receive any notifications for @mentioning yourself!

We recommend making sure at least two board members have enabled commenting, or creating a separate bot account for this purpose.
